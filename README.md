## FAQ

- ### Why create an entire repository just for a PDF?

    I don't just want to publish the PDF once it's finished. I want to be able to edit it constantly and easily. I can always step back and make changes, like if I want to rephrase certain sentences, I can. 
    
    And that way, I’m going to write it bit by bit, so that lets me break my work into segments.
    
    Plus, I can also update it based on current events. 
    
    It also has other advantages: if someone sees changes that need to be made, they can submit a pull request. And if someone wants to do the same for another PDF, they can fork my project and replace the .tex file.
